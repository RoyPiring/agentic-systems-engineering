# Architecture Decision Records

One file per decision: `ADR-NNN-kebab-case-title.md` (NNN = sequence, zero-padded).

Copy `_ADR-TEMPLATE.md` when creating a new ADR, then rename the new file (drop the leading underscore from the template copy only after you fill it — or duplicate and name `ADR-005-...` directly).

Summaries in root `architecture.md` link here; each ADR is the source of truth for rationale and alternatives.
